Practical Training Manual · QGIS 3.x LTR

Introduction

This manual is designed for GIS practitioners, students, and professionals who want to develop practical skills in QGIS — one of the world's most powerful and widely-used open-source Geographic Information System platforms. The course is structured progressively, starting from data acquisition and working through to professional map production.

By the end of this training, you will be able to:

  • Obtain, import, and manage geospatial datasets in multiple formats
  • Install and configure plugins to extend QGIS functionality
  • Understand and correctly apply coordinate reference systems (CRS/projections)
  • Apply cartographic styling and symbology to convey meaningful spatial information
  • Perform core spatial analysis operations
  • Edit vector features and attribute data
  • Produce professional print-ready maps using the QGIS Print Layout
📋 Note: All datasets used are thematically related to an urban planning and land use scenario, spanning Shapefile, GeoTIFF, CSV, and Excel formats.

Section 1: Obtaining Geospatial Data

1.1 Data Formats

FormatDescription
Shapefile (.shp)Most common vector format. Contains geometry + DBF attribute table.
GeoTIFF (.tif)Raster format encoding elevation, land cover, or satellite imagery.
CSV (.csv)Plain-text table. Used for tabular location data that QGIS can geocode.
Excel (.xlsx)Spreadsheet format for attribute-rich tabular data joinable to spatial layers.
GeoJSON (.geojson)Lightweight JSON-based vector format from web APIs and OSM exports.

Section 3: Importing Data

🗂️ Shapefile: Layer > Add Layer > Add Vector Layer (Ctrl+Shift+V) or drag from Browser Panel
🖼️ GeoTIFF Raster: Layer > Add Layer > Add Raster Layer (Ctrl+Shift+R)
📄 CSV as Points: Layer > Add Layer > Add Delimited Text Layer (Ctrl+Shift+T). Set X=Longitude, Y=Latitude, CRS=EPSG:4326
📊 Excel (.xlsx): Layer > Add Vector Layer > browse to .xlsx (loads as non-spatial table)
🌐 GeoJSON: Drag from Browser Panel onto canvas — CRS is embedded automatically
💡 Tip: Shapefile is a multi-file format (.shp, .dbf, .shx, .prj). Always copy ALL files with the same base name together.

Section 5: CRS & Projections

Geographic CRS
Uses degrees lat/lon. Best for global data.
Example: WGS 84 — EPSG:4326
Projected CRS
Uses metres. Best for area/distance analysis.
Example: UTM Zone 37N — EPSG:32637
✅ Best Practice: Always reproject data to a projected CRS (UTM) before performing area calculations, buffer operations, or distance measurements.

Set project CRS: Project > Properties > CRS. Reproject permanently: Right-click layer > Export > Save Features As > select target CRS.

Section 6: Styling & Symbology

Single Symbol — One style for all features. Properties > Symbology > Single Symbol
Categorised — Different colours per category (e.g. road type). Choose field > Classify
Graduated — Continuous numeric data (e.g. population density). Choose colour ramp > set classes
Raster — Singleband pseudocolor for DEM. Hillshade for terrain effect.
Labels — Properties > Labels > Single Labels. Add white text buffer for readability.

Section 8: Editing Vector Data

Enable Edit Mode: Click the ✏️ pencil icon or Layer > Toggle Editing. Save with Ctrl+S.
Edit Attributes: Open Attribute Table (F6) > click any cell in edit mode.
Digitise New Features: Add Point/Line/Polygon Feature from toolbar. Right-click to finish polygons/lines.
Field Calculator: Attribute Table > abacus icon. Compute new fields, e.g. $area / 1000000 for km².

Section 9: Print Layouts & Export

1. New Layout: Project > New Print Layout. Set page size (A3/A4, landscape).
2. Add Map: Add Item > Add Map. Draw rectangle. Set scale in Item Properties.
3. Add Elements: North Arrow, Scale Bar, Legend, Title Label, Inset Map.
4. Export: PDF (300 DPI for print) · PNG (150-200 DPI for web) · SVG (for Inkscape/Illustrator)

Section 10: Additional Topics

📷 Georeferencing
Layer > Georeferencer. Add ≥4 GCPs across the scanned image.
🌍 QuickOSM
Vector > QuickOSM. Query by key/value pair (e.g. highway=primary).
🔥 Heatmap
Layer Symbology > Heatmap renderer. Or Processing > Kernel Density Estimation.
⛰️ Hillshade DEM
Processing > Hillshade. Overlay at 60% transparency over styled DEM.

Section 12: Troubleshooting

IssueSolution
Layer not visible on canvasRight-click > Zoom to Layer. Check CRS and symbology colour.
CRS mismatch / layers misalignedCheck each layer CRS. Reproject mismatched layers permanently.
CSV imports but no points showVerify X/Y columns not swapped. Set CRS to EPSG:4326.
Plugin not appearingRestart QGIS. Check Plugins > Installed tab.
Raster all one colourProperties > Symbology > Load Min/Max values > Apply.
